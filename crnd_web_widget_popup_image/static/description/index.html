CRnD Widget Popup Image


CRnD Widget Popup Image is a module developed by the Center of Research & Development company. This widget allows you to popup images from the binary fields. It is available on the form and tree views.

How it works:

  • Define a widget on the form view: 
    
<group>
    <field name="image_test" widget="image_popup" class="oe_avatar"/>
</group>
  • If necessary, define a widget on the tree view: 
    
<tree>
    <field name="test_field"/>
    <field name="image_test" widget="image_popup" class="ot_image"/>
</tree>

    The ot_image CSS class resizes the image for the tree view (you can use your own class.)

As a result:

  • Form view


  • Tree view


  • Popup


  • Edit mode

    In the edit mode, it works as a standard image widget.
